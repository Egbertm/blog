1、字符串的旋转

对于字符串的旋转，大多最开始想到的办法蛮力法，然而对字符串的整体旋转我们可以采用
三步旋转法如：对于字符串abcdef,若要将def移动到abc的前面怎么在O(N)的时间复杂度下
完成。具体步骤如下：分别对abc和def字符串反转，得到cba和fed,在把整体的得到的字符
串反转就得到了defabc。如下：
``` java
    public static String LeftRotateString(String str,int m){
        str = ReverseString(str,0,m-1);
        str = ReverseString(str,m,str.length()-1);
        str = ReverseString(str,0,str.length()-1);
        return str;
    }
    public static String ReverseString(String str,int from,int to){

        char []s = str.toCharArray();
        while(from<to){
            char t = s[from];
            s[from++] = s[to];
            s[to--] = t;
        }
        return new String(s);
    }

```

``` java
    public String replaceSpace(StringBuffer str) {
```

4、重复字符的压缩

题目：对一给定的字符串，把连续的相等字符压缩并统计它的次数,如：字符串aaabbcbb压
缩后为a3b2c1b2这种形式。具体如下：

``` java
    public String compress(String str){
        if(str == null || str.isEmpty())  
            return null;  
        StringBuffer mystr = new StringBuffer();  
        char last = str.charAt(0);  
        int count = 1;  
        for(int i = 1;i<str.length();i++){  
            if(str.charAt(i) == last){  
                count++;  
            }else{  
                mystr.append(last);  
                mystr.append(count);  
                last = str.charAt(i);  
                count= 1;  
            }  
        }  
        mystr.append(last);  
        mystr.append(count);  
        return mystr.toString();
    }
```
